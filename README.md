# Arch-i3-Dotfiles

## Requirements
* Arch Linux
* git (pacman -S git)

## Install

`git clone https://github.com/akshayrkapadia/Arch-i3-Dotfiles.git`
`cd Arch-i3-Dotfiles`
`pacman -S - < packages.txt`
`./install`
`mv Arch-i3-Dotfiles /home/akshay/`
`cd /home/akshay/Arch-i3-Dotfiles/`
`./configurei3`
`exit`
`reboot`
